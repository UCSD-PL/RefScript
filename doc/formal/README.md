## Building instructions 

### Getting ott

Ott is available
[here](http://www.cl.cam.ac.uk/~pes20/ott/ott_distro_0.24.tar.gz).


### Building ott

Requires recent ocaml installation. Then just run `make` and add `bin/ott` to your path.
